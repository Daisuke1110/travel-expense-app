# プロダクト概要

## コンセプト

海外旅行によく行くユーザー向けに、現地通貨での支出をカンタンに記録し、
リアルタイムのレートで日本円換算して把握できるアプリ。

友達と一緒の旅行では、共有のトリップを使って全員の支出をまとめて管理できる。

- 形式: PWA + AWS サーバレス
- 対象: iPhone をメインに想定（Safari + ホーム画面追加）
- 利用規模: 個人および友人数名程度を想定し、AWS コストは月数百円以内を目標とする

## ユースケース

- 自分一人で海外旅行に行くとき:

  - 自分専用のトリップを作成
  - 支出を現地通貨で記録
  - 合計と日本円換算をすぐに確認できる

- 友達と海外旅行に行くとき:

  - 共有トリップを作成
  - 参加メンバー全員が支出を登録
  - 後で「誰がどれだけ払ったか」を振り返るベースにできる（将来割り勘機能も）

## ログイン後の挙動

- ユーザーがログインすると、最初に以下が表示される:

  - 「自分が作成したトリップ一覧」
  - 「友達と共有しているトリップ一覧」

- ユーザーはそこから 1 つのトリップを選択して、支出画面に入る。

## 通貨とレート

- 支出金額は、ユーザーが現地通貨で手入力する。
- 日本円換算は、以下の方針で行う:

  - MVP:
    - トリップごとに「基準通貨」と「円レート（1 単位あたりの円）」を手入力し、
      支出登録時に `amount * rate_to_jpy` で日本円換算する。
  - 将来:
    - 外部の為替レート API からレートを取得し、自動で `rate_to_jpy` を更新する。

- 将来的には、レシート画像から金額や通貨を読み取る OCR 機能を追加予定。

## 今後の拡張イメージ

- レシート撮影 → 金額と通貨を自動抽出
- 割り勘機能（誰がいくら立て替えたか）
- 日別/カテゴリ別の可視化（グラフ）
- オフライン中の入力とオンライン時の自動同期
- 為替レート API と連携したレート自動取得

## MVP の範囲（最初に実装する機能）

- ログイン（暫定的な簡易認証、将来的に Cognito へ移行）
- ログインユーザーのトリップ一覧表示（自分のトリップ / 共有トリップ）
- トリップ作成（基準通貨・レートを含む）
- トリップごとの支出の登録・一覧表示
- 現地通貨合計と日本円換算合計の表示
- PWA として iPhone のホーム画面に追加して利用できる
