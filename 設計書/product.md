# プロダクト概要

## コンセプト
海外旅行での支出を現地通貨で簡単に記録し、リアルタイムの換算レートで日本円合計を把握できるアプリ。  
友達との旅行では共有トリップを使って全員の支出をまとめて管理できる。  
- 形態: PWA + AWS サーバレス
- 対象: iPhone を主想定（Safari + ホーム画面追加）
- 利用規模: 個人および友人数名。AWS コストは月数百円〜千円台を目標。

## ユースケース
- 個人旅: 自分用トリップを作成し、現地通貨で支出記録。合計と円換算を確認。
- 友人旅: 共有トリップを作成し、参加メンバー全員が支出登録。誰がいくら払ったかのベースにする（割り勘機能は将来）。

## 起動時（=ログイン後）挙動
- MVP ではログイン画面なし。デバッグヘッダ `X-Debug-User-Id` で user_id を指定して起動。
- 起動時に「自分が作成したトリップ一覧」「共有トリップ一覧」を取得し、ユーザーが 1 つを選択して支出画面に入る。

## 通貨とレート
- 支出金額はユーザーが現地通貨で手入力。
- 日本円換算の方針
  - MVP:
    - トリップごとに「基準通貨」と「レート（1 単位あたりの円）」を手入力し、`amount * rate_to_jpy` で換算。
    - **基準通貨（base_currency）はトリップ作成後は変更不可。** 変更要求時は 400 を返す（backend.md で明記）。レートのみ変更可。
    - レートを更新しても支出レコードの金額は変更せず、表示時に最新レートで換算。レート履歴は保持しない。
  - 将来:
    - 外部為替レート API からレートを取得し自動更新。
    - レート履歴を保持し、当時レートでの再現や差分表示に対応。
- 将来はレシート画像から金額/通貨を読む OCR を追加する構想。

## 今後の拡張イメージ
- レシート撮影 ↔ 金額・通貨自動抽出
- 割り勘機能（誰がいくら立て替えたか）
- 日別/カテゴリ別の可視化（グラフ）
- オフライン入力とオンライン時の自動同期
- 為替レート API と連携したレート自動取得・履歴保持

## MVP の範囲（最初に実装する機能）
- ログイン: 簡易認証（ヘッダ指定）。将来 Cognito へ移行。
- トリップ一覧表示（自分/共有）、トリップ作成（基準通貨・レートを含む）
- 共有トリップのメンバー追加（オーナーのみ、`POST /trips/{tripId}/members` で user_id 手入力。メール招待は将来）
- トリップごとの支出登録・一覧表示
- 現地通貨合計と日本円換算合計の表示（常に最新レートで換算する旨を UI で示す）
- PWA として iPhone のホーム画面に追加して利用できる
